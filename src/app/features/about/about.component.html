Características

Funcionalidades principales de la plataforma Userly

Gestión de Usuarios

Visualización completa de hasta 40 usuarios obtenidos de la Random User API con información detallada de cada perfil

Búsqueda Avanzada

Búsqueda en tiempo real por nombre, email o país con filtrado instantáneo de resultados

Vista Detallada

Acceso a información completa de cada usuario incluyendo datos personales, ubicación y contacto

Paginación

Sistema de paginación con 10 usuarios por página para una navegación eficiente y organizada

Reporte en Excel

Exportación de datos filtrados a formato Excel (.xlsx) con información de usuarios para análisis externo

Diseño Responsive

Interfaz adaptativa que funciona perfectamente en dispositivos móviles, tablets y escritorio
